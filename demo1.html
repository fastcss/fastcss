#
What is fastcss

When I discovered sass I've seen the potential of class generation. I made some utilities with hardcoded configuration which were very helpfull in my projects. After a while I had this bright idea to transform it into a fully customizable pure sass-generated css framework.

#
Installation

Yarn
yarn add fastcss
React
yarn add fastcss-react
Vue
yarn add fastcss-vue
CDN
<\script\>\https://cdn/fastcss.css<\/script>
